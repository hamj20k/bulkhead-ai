# Code of Conduct

Bulkhead welcomes contributors who are direct, curious, and respectful.

## Expected behavior

- Treat other contributors with respect.
- Critique code, claims, and tradeoffs without attacking people.
- Assume good intent, but accept corrections when impact differs from intent.
- Keep security discussions factual and responsible.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Publishing private vulnerability details before a fix or disclosure plan.
- Spamming issues or pull requests.
- Misrepresenting Bulkhead as a complete prompt-injection solution.

## Enforcement

Maintainers may edit, hide, or delete comments; close issues; reject pull
requests; or block contributors who violate this code. If there is a problem,
contact the maintainer privately.
